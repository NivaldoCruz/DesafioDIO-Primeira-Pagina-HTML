Surgimento da internet

(Voltar)

Após a criação dos primeiros computadores a tecnologia continuou avançando até que fosse estabelecido o conceito de redes de computadores.

Por volta da década de 60 o conceito da rede mundial de computadores conhecida como internet seria criado por meio da rede chamada de arpanet, o intuito da internet era que a mesma fosse utilizada para fins militares e de pesquisa.

Em meados da década de 90, Tim berners-Lee seria responsável pela criação da world wide web(WWW), a rede mundial de computadores como é conhecida, criando assim a internet como conhecemos hoje, possibilitando o acesso a informação em computadores a longas distancias.

A internet foi projetada para ser acessada pelo meio de softwares chamados de navegadores como Internet Explorer, Google Chrome, Mozilla firefox, entre outros.

A principal ferramenta para desenvolvimento web na época era o HTML, os navegadores foram projetados para interpretar o HTML e transmitir a informação ao usuário de forma visual.

Princípios do HTML

(Voltar)

Ao contrário do que muitos acreditam o HTML(Hypertext Markup Language) ou linguagem de marcação de hipertexto, não é uma linguagem de programação, mas sim uma linguagem de marcação de texto amplamente utilizada no desenvolvimento web para criação de websites.

Com a criação do World Wide Web Consortium(W3C), foi gerada a especificação formal da linguagem HTML, a fim de que sua utilização fosse padronizada e facilitada independente de qual navegador executa-se um arquivo HTML.

Principais tags do HTML

(Voltar)

A utilização do HTML é feita pelo uso das tags. As tags permitem formatações na forma que o texto é mostrado, organização das informações, entre outras funções.

Veja algumas das principais tags HTML abaixo

Head: Age como o cabeçalho da página, sendo possível definir o titulo da mesma por exemplo.

Body: Funcionando como o corpo da página, dentro desta tag é possível definir os conteúdos exibidos pelo website

h(1,2,3,4,5,6): A tag h permite a criação de sub títulos na página de diferentes hierarquias.

p: Tag responsável pela criação de parágrafos onde a maioria do texto de uma página se encontra.

ol/ul: Responsáveis pela criação de listas e auxiliadas pela tag li que permite a listagem dos itens nessas listas.
